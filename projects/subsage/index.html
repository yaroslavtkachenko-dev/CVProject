SubSage

Розумний менеджер підписок для iOS

Елегантний та інтуїтивно зрозумілий додаток, який допомагає легко керувати всіма вашими щомісячними та щорічними підписками. Побудований з використанням найсучасніших технологій iOS розробки.

SwiftUI Core Data WidgetKit Charts UserNotifications
Можливості Скріншоти

Ключові функції

Керування підписками

Легко додавайте, редагуйте та відстежуйте всі ваші підписки в одному місці.

Розумні нагадування

Отримуйте сповіщення про майбутні платежі, щоб ніколи не пропустити важливу дату.

Аналітика витрат

Візуалізуйте ваші витрати з допомогою інтерактивних графіків та діаграм.

В розробці
Віджети для iOS

Швидкий доступ до інформації про підписки прямо з головного екрану.

Приватність даних

Всі дані зберігаються локально на вашому пристрої за допомогою Core Data.

В розробці
iCloud Синхронізація

Синхронізуйте ваші підписки між всіма вашими пристроями Apple через iCloud.

Скріншоти

Технічні деталі

Архітектура та технології
  • SwiftUI: Сучасний декларативний фреймворк для створення інтерфейсів
  • Core Data: Надійне локальне зберігання даних з підтримкою складних запитів
  • WidgetKit: Інформативні віджети для головного екрану iOS
  • UserNotifications: Система локальних сповіщень про майбутні платежі
  • Charts: Інтерактивна візуалізація фінансових даних
Особливості розробки
  • MVVM архітектурний патерн для чистого та тестованого коду
  • Адаптивний дизайн для різних розмірів екранів iPhone
  • Підтримка темної та світлої тем iOS
  • Локалізація інтерфейсу українською та англійською мовами
  • Дотримання Human Interface Guidelines Apple
