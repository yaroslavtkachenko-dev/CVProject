Technical Details

Architecture and Technologies
  • SwiftUI: Modern declarative framework for building user interfaces
  • Core Data: Reliable local data storage with support for complex queries
  • WidgetKit: Informative widgets for iOS home screen
  • UserNotifications: Local notification system for upcoming payments
  • Charts: Interactive visualization of financial data
Development Features
  • MVVM architectural pattern for clean and testable code
  • Adaptive design for different iPhone screen sizes
  • Support for iOS dark and light themes
  • Interface localization in Ukrainian and English
  • Compliance with Apple Human Interface Guidelines
